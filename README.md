# pyfun_morse
Morse code translator

## Features
- [x] Decodes Morse code to text
- [x] Encodes text to Morse code
- [ ] Accepts as input
    - [x] string
    - [ ] text file
- [ ] Produces output as
    - [x] string
    - [ ] text file
- [x] Configurable characters for dots and dashes
- [x] Validates all inputs
- [ ] CLI
